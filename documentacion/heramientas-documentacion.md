## Herramientas de documentacion

* Se recogen 3 heramientas diferentas para la generacion de documentacion
---

## 1. Github y github action
Con github generamos documentacion de manera manual, pero con Github Action podemos automatizar este proceso.
## 2.JSdoc

con jsdoc generamos automaticamente la documentacion en lenguaje de javascript

## 3.Docstring

con Docstring podemos generar documentacion si estamos trabajando con Python
